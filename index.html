What is CI/CD?

CI/CD is a short cut for Continuous Integration and Continuous Deployment. CI/CD is a way to automate the process of building and deploying software. When developers push their code to a repository like GitHub, the system can automatically run checks and deploy the application. This helps teams release updates faster and makes the development process easier and more reliable.

My Pipeline

When I push my code to the main branch on GitHub, a workflow starts automatically. GitHub Actions checks if the required files exist in the repository. If everything is correct, the workflow uploads the files and deploys the website to GitHub Pages so it becomes available online.
